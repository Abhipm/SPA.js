Examples

Base

Let's start with a base page to build SPA.

<!DOCTYPE html>
<html>
  <head>
    <title>SPA Boilerplate</title>
  </head>

  <body>

    <h1>Hello, SPA!</h1>

    <!-- jQuery, lodash and handlebars - necessary for SPA. (from CDN repository) -->
    <script src="//cdn.jsdelivr.net/jquery/2.2.3/jquery.min.js"></script>
    <script src="//cdn.jsdelivr.net/lodash/4.12.0/lodash.min.js"></script>
    <script src="//cdn.jsdelivr.net/handlebarsjs/4.0.5/handlebars.min.js"></script>

    <!-- Include SPA (from local repository) -->
    <script src="../../src/spa.min.js"></script>
  </body>
</html>

Example-1

Local data with inline template

Local Data

var myPost = {
  title: "My 1st Blog Post!",
  author: {
    id: 101,
    name: "Jack O Neil"
  },
  body: "My 1st post. Hurray!"
};

Inline Template

<div data-url="local:myPost"
     data-render>
  <h1>{{title}}</h1>
  <h2>By {{author.name}}</h2>

  <div>
    {{body}}
  </div>
</div>

Results in

<div data-url="local:myPost"
     data-render>
  <h1>My 1st Blog Post!</h1>
  <h2>By Jack O Neil</h2>

  <div>
    My 1st post. Hurray!
  </div>
</div>

Example-2

Local data with internal template (hidden within the page as template collection)

Local Data

var myPost = {
  title: "My 2nd Blog Post!",
  author: {
    id: 102,
    name: "Jack O Neil"
  },
  body: "My 2nd post. Hurray!"
};

Internal Template

<div data-url="local:myPost" 
     data-template="#tmplPost"
     data-render>
</div>

<section style="display:none">
  <div id="tmplPost">
    <h1>{{title}}</h1>
    <h2>By {{author.name}}</h2>

    <div>
      {{body}}
    </div>
  </div>
</section>

Results in

<div data-url="local:myPost"
     data-template="#tmplPost"
     data-render>
  <h1>My 2nd Blog Post!</h1>
  <h2>By Jack O Neil</h2>

  <div>
    My 2nd post. Hurray!
  </div>
</div>

Example-6

Render on click instead of on page load.

External Data: api/myPost.json

{
  title: "My 4th Blog Post!",
  author: {
    id: 104,
    name: "Jack O Neil"
  },
  body: "My 4th post. Hurray!"
}

External Template: templates/tmplPost.html

  <h1>{{title}}</h1>
  <h2>By {{author.name}}</h2>

  <div>
    {{body}}
  </div>

Base Page

<div id="divViewPost"
  data-url="api/myPost.json" 
  data-template="templates/tmplPost.html"
  >
</div>

<button
  onclick="spa.render('#divViewPost');"
  >Load</button>

Note: No "data-render" attribute in render container.

Results in

<div id="divViewPost"
  data-url="api/myPost.json" 
  data-template="templates/tmplPost.html"
  >
  <h1>My 4th Blog Post!</h1>
  <h2>By Jack O Neil</h2>

  <div>
    My 4th post. Hurray!
  </div>
</div>

<button
  onclick="spa.render('#divViewPost');"
  >Load</button>

Example-7

Dynamic Data-URL with params.

External Data: api/myPost.json

{
  title: "My 4th Blog Post!",
  author: {
    id: 104,
    name: "Jack O Neil"
  },
  body: "My 4th post. Hurray!"
}

External Template: templates/tmplPost.html

  <h1>{{title}}</h1>
  <h2>By {{author.name}}</h2>

  <div>
    {{body}}
  </div>

function to render

function loadMyPost() {
  spa.render( { 
    target : "#divViewPost",
    dataUrl: "api/myPost.json",
    dataParams: { pid: 12345 }
    } );
}

  data-template="templates/tmplPost.html"
  >
</div>

  onclick="loadMyPost();"

  >
  <h1>My 4th Blog Post!</h1>
  <h2>By Jack O Neil</h2>

  <div>
    My 4th post. Hurray!
  </div>
</div>

<button
  onclick="loadMyPost();"
  >Load</button>

Example-8

Inner Templates.

External Data: api/myPost.json

{
  title: "My 4th Blog Post!",
  author: {
    id: 104,
    name: "Jack O Neil"
  },
  body: "My 4th post. Hurray!"
}

External Data: api/relPost.json

{
  "pid" : "12345",
  "rposts": [ 
      { "title": "Simplified Ajax",
        "date": "3 Nov 2003"},
      
      { "title": "JS Tool Box",
        "date": "3 May 2004"},
      
      { "title": "Web site vs app",
        "date": "6 Aug 2005"}
    ]
}

function to render

function loadMyPost() {
  spa.render( { 
    target : "#divViewPost",
    dataUrl: "api/myPost.json",
    dataParams: { pid: 12345 }
    } );
}

Base Page

<button
  onclick="loadMyPost();"
  >Load</button>

<div id="divViewPost"
  data-template="templates/tmplPost.html"
  >
</div>

External Template-1: templates/tmplPost+Rel.html

<h1>{{title}}</h1>
<h2>By {{author.name}}</h2>

<div>
  {{body}}
</div>

<h3>Other Posts:</h3>
<ul
  data-template="templates/tmplRel.html" 
  data-url="api/relPost.json" 
  data-render>
</ul>

External Template-2 (to use in 1): templates/tmplRel.html

{{#each rposts}}
   <li>{{title}} - {{date}}</li>
{{/each}}

Results in

<button
  onclick="loadMyPost();"
  >Load</button>

  <div id="divViewPost"
  data-template="templates/tmplPost.html"
  >
  <h1>My 4th Blog Post!</h1>
  <h2>By Jack O Neil</h2>

  <div>
    My 4th post. Hurray!
  </div>

  <h3>Other Posts:</h3>
  
  <ul
    data-template="templates/tmplRel.html"
    data-url="api/relPost.json"
    data-render>
    <li>Simplified Ajax - 3 Nov 2003</li>
    <li>JS Tool Box - 3 May 2004</li>
    <li>Web site vs app - 6 Aug 2005</li>
  </ul>

  </div>

API Doc

spa.render

Usage:

spa.render(targetContainerId, renderOptions)

spa.render(renderOptions)

Example:

spa.render( '#containerView', { dataUrl: 'api/module/data.json', template: 'templates/view.html' } )

spa.render( { target: '#containerView', dataUrl: 'api/module/data.json', template: 'templates/view.html' } )
Options HTML Tag Attributes Comments/Examples
- data-render

to auto render the container on page load.

<div ... ... data-render></div>

dataModel data-model

data model path on return JSON object of dataUrl

Example: if dataUrl returns:

{
  keyA: {

    keyB: {

      keyC: {
      ...
      ...
      }
    }
  }
}

and the template consumes the value of "KeyC"


{ dataModel: "keyA.keyB.KeyC", ... ... }
<div data-model="keyA.keyB.KeyC" ... ... ></div>

Get Started

 

Download SPA.js

The fastest way to get SPA.js is to download the compiled and minified versions of the library.

Download SPA.js

 

Additional downloads

SPA's hard dependencies are jQuery (>= 1.9.0), LoDash (>= 2.4.1) and Handlebars (>= 1.3.0)

  • jQuery
  • Lodash
  • Handlebars

 

Quick Start

Make use of a super basic HTML boilerplate to get started. Copy and paste the HTML from below to get started with a bare bones SPA document.


  <!DOCTYPE html>
  <html>
    <head>
      <title>SPA Boilerplate</title>
    </head>

    <body>

      <h1>Hello, SPA!</h1>

      <!-- jQuery, lodash and handlebars - necessary for SPA. (from CDN repository) -->
      <script src="//cdn.jsdelivr.net/jquery/2.2.3/jquery.min.js"></script>
      <script src="//cdn.jsdelivr.net/lodash/4.12.0/lodash.min.js"></script>
      <script src="//cdn.jsdelivr.net/handlebarsjs/4.0.5/handlebars.min.js"></script>

      <!-- Include SPA (from local repository) -->
      <script src="xlib/spa/spa.min.js"></script>
    </body>
  </html>

See Examples for more options to build SPA with local/server-side data and templates.

Get Started

 

Download SPA.js

The fastest way to get SPA.js is to download the compiled and minified versions of the library.

Download SPA.js

 

or use bower to download with it's dependencies 
bower install spa.js

 

Additional downloads

SPA's hard dependencies are jQuery (>= 1.9.0) and Handlebars (>= 1.3.0)

  • jQuery
  • Handlebars

 

Quick Start

Make use of a super basic HTML boilerplate to get started. Copy and paste the HTML from below to get started with a bare bones SPA document.


  <!DOCTYPE html>
  <html>
    <head>
      <title>SPA Boilerplate</title>
    </head>

    <body>

      <h1>Hello, SPA!</h1>

      <!-- jQuery and handlebars - necessary for SPA. (from CDN repository) -->
      <script src="//cdn.jsdelivr.net/npm/jquery/dist/jquery.min.js"></script>
      <script src="//cdn.jsdelivr.net/npm/handlebars/dist/handlebars.min.js"></script>

      <!-- Include SPA (from CDN repository) -->
      <script src="//cdn.jsdelivr.net/gh/sucom/SPA.js/dist/spa-bundle.min.js"></script>
    </body>
  </html>

See Examples for more options to build SPA with local/server-side data and templates.
